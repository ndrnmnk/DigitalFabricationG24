# DigitalFabricationG24

This is a repository of the project students of group 24 made during `Digital Fabrication` course in University of Oulu in 2026.

### X3mist
X3mist is a pager based on a Raspberry pi Pico. It uses RFM95w and WiFi radio module for message exchange.  
The messages are encrypted using AES-256-CBC, and the keys are exchanged through IR LED and receiver.

Other features include a 12 button keyboard and a buzzer.
